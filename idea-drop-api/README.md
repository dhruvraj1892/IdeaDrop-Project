# 💡 Idea Drop

A full-stack MERN application where users can share, explore, and manage innovative ideas.

## Features

- User Authentication (JWT + Refresh Tokens)
- Create, Read, Update and Delete Ideas
- Protected Routes
- Responsive UI
- RESTful API
- MongoDB Atlas Database
- Secure Password Hashing (bcrypt)

## Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

**Authentication**
- JWT
- bcrypt
