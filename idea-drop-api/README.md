# 💡 Idea Drop

A full-stack MERN application where users can share, explore, and manage innovative ideas with secure authentication.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Refresh Token Support
- Protected Routes
- Create, Read, Update, and Delete Ideas
- Responsive User Interface
- RESTful API
- Secure Password Hashing with bcrypt
- MongoDB Database Integration
- Error Handling & Input Validation

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcrypt

### Tools
- Git
- GitHub
- Postman
- VS Code
- Nodemon

## 📦 Installation

```bash
git clone https://github.com/dhruvraj1892/idea-drop.git
cd idea-drop
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the backend folder:

```env
PORT=
MONGO_URI=
JWT_SECRET=
JWT_REFRESH_SECRET=
```
